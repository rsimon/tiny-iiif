# tiny.iiif

A minimal Docker-based IIIF server. Backed by Cantaloupe, NGINX and a tiny admin GUI.

## Prerequisites

- Docker 24.0+
- Docker Compose 2.x
- NodeJS 20+
- npm 11+

## Development

- Run `docker compose up` to start the backend
- `cd gui`
- `npm install`
- `npm start`
- Open <http://localhost:4321>
